I have updated my PCGen data files to work with the newest version of PCGen.  I'd like everyone to download it and use the newer versions instead of the old 2.7.3 so that we're all on the same proverbial page. The new PCGen is faster, and will very soon be supporting stuff like d20 Modern that I know we all want. :)  To use my data files, download and install PCGen as normal (the download site is here), and then download and unzip my file.  You should see a folder called "data."  You need this folder in addition to the data folder that comes with the PCGen download.  You need them both.  Drop both data folders into the main PCGen folder. When you put the second one in, it'll ask you if you're sure you want to move it, that there is already a folder called data and that files with the same name will be overwritten.  Say "yes to all."  And presto, you're in good shape.

Lastly, I've officially launched my Planescape site, The Kriegstanz , today.  I've put a god-awful ton of work into it, and I'm very happy with the outcome.  I've received over one thousand hits today, so I'm considering it a success.  Those are pretty good first day numbers, if I do say so myself. :)  Anyway, I'd love to hear what y'all think about it, so drop by and check it out if you have the time.
