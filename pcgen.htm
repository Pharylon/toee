PCGen Data Files

 

PCGen Installation - This is a full version of PCGen, along with my data files, which include custom campaign stuff as well as WotC products no longer in current PCGen releases.  This download is version 4.2.7, and contains everything you need.  Just download and unzip.

PCGen Data Files - Alternately, if you are using a later version (there are no later versions as of this writing, but I doubt I'll update the full version above very often), you can download just my data files here.  To use my data files, download and install PCGen as normal (the download site is here), and then download and unzip my file.  You should see a folder called "data."  You need this folder in addition to the data folder that comes with the PCGen download.  You need them both.  Drop both data folders into the main PCGen folder. When you put the second one in, it'll ask you if you're sure you want to move it, that there is already a folder called data and that files with the same name will be overwritten.  Say "yes to all."  And presto, you're golden.
